Stage Details

1

Raw Data

Image dataset in PNG format with accompanying JSON annotation files containing damage bounding regions and classes.

2

Preprocessing

Using Jupyter notebooks, the data is cleaned, converted to YOLO format, and split into train / validation / test sets.

3

Training

We use the Ultralytics framework to train a segmentation model on the processed data, tracking metrics and weights.

4

Inference

The trained YOLOv26-seg model analyzes uploaded images and produces precise segmentation masks of damaged regions.

5

Streamlit UI

Interactive web interface built with Streamlit that lets the user upload images and view analysis results and the final report.
